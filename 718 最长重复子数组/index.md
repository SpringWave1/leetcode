# 题目
如题, 例如 
输入:
A: [1,2,3,2,1]
B: [3,2,1,4,7]
输出: 3
解释: 
长度最长的公共子数组是 [3, 2, 1]。
# 思路
1. 暴力解法
   1. 枚举公共起点，然后计算最长数组
2. 动态规划
   1. 定义：dp[i][j] 定义了字符串 s1 前 i 个字符和 s2 前 j 个字符所能组成的最大字符串
      1. 例如 dp[2][2] 所能构成的最长字符串 = dp[1][1] + 当前字符是否相等
      2. 如果不相等 dp[2][2] 为 0
   2. 公式
      1. if s1[i] == s2[j]
         1. dp[i][j] = dp[i-1][j-1] + 1
      2. else:
         1. dp[i][j] = 0

# 类比
最长公共子序列
- 和最长公共子串的最大区别在于不需要连续
  - 就算不相等当前也不会置位为 0 
  - 并且在递归中，只需要看相邻的值 （i-1） or (j-1)即可 
  - if s1[i] == s2[j]
    - dp[i][j] = dp[i-1][j-1] + 1
  - else:
    - dp[i][j] = max(dp[i-1][j], dp[i][j-1])
