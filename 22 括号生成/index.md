# 题目描述
数字 n 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 有效的 括号组合。
>
```
输入：n = 3
输出：[
       "((()))",
       "(()())",
       "(())()",
       "()(())",
       "()()()"
     ]
```
# 思路
n = 3 的情况可以分解成 n = 2 的情况，递归。
设定两个值 left 和 right，作递归。
四种情况递归结束
- base case: left == 0 and right == 0 括号用完，递归终点
- bad case：
  - 超量
    - left < 0 没有左括号
    - right < 0
  - left > right 
    - 因为我们从左括号开始，如果left > right ，右边不可能有 available case
